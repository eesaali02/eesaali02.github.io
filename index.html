Projects

Asteroid Apocalypse

In this fast-paced action game, you play as a spaceship pilot navigating through an asteroid field. Your mission is to shoot as many asteroids as possible while avoiding collisions. The game was designed so that, as you progress through the 3 levels, the asteroids become more numerous and difficult to evade. Extra care was put into consideration when designing the mechanics to ensure intuitive controls and smooth gameplay. Built using MonoGame and C#, this game showcases the power of these technologies in creating immersive and engaging experiences.

Canada Quiz Game

Test your knowledge of Canada with this fun and challenging trivia game! Created with Windows Form Applications in Visual Studio 2022, this game features a range of questions about Canada's history, culture, geography, and more. It's intuitive design allows for a smooth transition between questions, allows one hint, and a 50/50 button to eliminate half of the options. Try it out to see if you can acheive a perfect score!

Static Web Page

This is a static webpage created using HTML and CSS. The page is a personal branding site for myself. The page is uniqley desinged and incorporates multiple pages within the site.

Horse Race

In this game, 3 horses race around a circular path where their timing and speed are recorded. Developed in Visual Studio 2022 using C#, this game randomizes the increase in arc length per interval for each horse to create a new race result each time.
